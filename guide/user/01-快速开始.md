# 01 快速开始：从零到预览（10 分钟）

本页用“可复制粘贴”的方式带你跑通一次完整链路：初始化站点 → 写一篇内容 → 构建 → 本地预览 → 准备部署。

## 你将获得什么

- 一个可以本地预览的静态站点（输出在 `dist/`）
- 一个最小可用的 `site.yaml`（后续所有功能都从它扩展）
- 一套最常用的命令行命令（build/preview/doctor/clean）

## 前置条件

- 已安装 .NET（项目目标为 .NET 10；如果你使用仓库源码方式运行，需要本机具备对应 SDK）
- 会使用命令行（PowerShell / bash）
- 了解 YAML/Markdown 的基本语法

## 路径 A：直接跑通仓库内示例站点（推荐）

示例站点位于：`examples/starter/`，自带内容、主题、以及多语言 / modules 等变体配置，适合用来确认环境是否正确。

### 1）构建并自检（doctor）

在仓库根目录执行：

```bash
dotnet build sitegen.slnx -c Release
dotnet run --project src/SiteGen.Cli -c Release -- doctor --config examples/starter/site.yaml
```

如果 doctor 报错，优先看：[14-故障排查](./14-故障排查.md)（以及开发者版 doctor 说明：[guide/dev/doctor](../dev/doctor.md)）。

### 2）构建站点（build）

```bash
dotnet run --project src/SiteGen.Cli -c Release -- build --config examples/starter/site.yaml --clean --site-url https://example.com
```

构建产物会输出到示例配置中的 `build.output`（默认 `examples/starter/dist/`）。

### 3）本地预览（preview）

```bash
dotnet run --project src/SiteGen.Cli -c Release -- preview --dir examples/starter/dist --port auto
```

控制台会打印一个本地 URL，用浏览器打开即可。

## 路径 B：创建你自己的站点（Markdown 模式）

如果你是要开始一个真实网站工程，建议在一个新目录里执行 `create`，它会生成基础目录结构与默认配置。

### 1）创建站点骨架

```bash
dotnet run --project src/SiteGen.Cli -c Release -- create my-site
```

你将得到类似结构（示意，实际以脚手架输出为准）：

```text
my-site/
  site.yaml
  content/
  layouts/    # 或 themes/（视脚手架与主题选择）
  assets/
  static/
```

### 2）编辑最小配置（site.yaml）

最小可用的 `site.yaml`（Markdown 站点）长这样：

```yaml
site:
  name: my-site
  title: My Site
  baseUrl: /
  language: zh-CN
  timezone: Asia/Shanghai
content:
  provider: markdown
  markdown:
    dir: content
    defaultType: page
build:
  output: dist
  clean: true
theme:
  layouts: layouts
  assets: assets
  static: static
  name: alt
logging:
  level: info
```

更完整的字段解释与默认值见：[04-配置-site-yaml](./04-配置-site-yaml.md)。

### 3）写第一篇内容（content/hello-world.md）

```markdown
---
type: page
title: Hello World
slug: hello-world
tags: [demo, first]
summary: 这是我的第一篇页面
---

# Hello World

如果你能看到这段文字，说明构建与渲染链路已经跑通。
```

### 4）自检（doctor）

在站点目录执行：

```bash
dotnet run --project ../src/SiteGen.Cli -c Release -- doctor --config site.yaml
```

### 5）构建与预览

```bash
dotnet run --project ../src/SiteGen.Cli -c Release -- build --config site.yaml --clean --site-url https://example.com
dotnet run --project ../src/SiteGen.Cli -c Release -- preview --dir dist --port auto
```
